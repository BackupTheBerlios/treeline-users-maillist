Sounds like you don't have the unstable tree enabled.

Use 'fink configure' from a command line, answer 'yes' to the question  
about using unstable (you should be able to just hit the carriage  
return for all other questions). When fink configure finishes, so  
'fink selfupdate', which will retrieve all the package descriptions  
for  unstable files. Then you should be able to 'fink install treeline'.

In finkcommander, use the Preferences dialog, fink tab, to check the  
box about using unstable packages. After leaving Preferences, use the  
Source menu -> selfupdate to do the package list update. And treeline  
should appear in the list.

If you don't have qt3 installed, doing the installation will take a  
while, since qt3 is huge, and you need it to build treeline. Pyqt will  
also take a noticeable amount of time to build and install.
