I created a plugin that automates a few things that I wanted to do in
Treeline. It is compatible with the new stable version (1.2.0) of
treeline, not any prior stable version (such as 1.0.2). I haven't
tried it on any unstable release.

My plugin provides three little utilities. The first takes selected
text and attempts to use it as a URL and open a webbrowser. The second
inserts a tab character in an edit window. The third creates a regular
expression search (as opposed to the current keyword search).

Constructive criticism and/or bug reports welcome (although I may not
do anything with them ;-).
