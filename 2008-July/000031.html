TreeLine 1.2.0 compile and run very well on Mac OS X 10.5.4 (cs_CZ 
locale setting), with only small usual corrections(same as in older 
unstable betas) in treeline.py.

How to install:
1. Download and install qt-mac-opensource-4.4.0.dmg.
2. Download and configure-make-make install sip-4.7.6.tar.gz
3. Download and configure-make-make install PyQt-mac-gpl-4.4.2.tar.gz
4. Download a decompress treeline-1.2.0.tar.gz
4a. Edit source code of "treeline.py" file, or apply diff.

Here is a diff:

4b. Install TreeLine using standard install.py
5. Start treeline from terminal. Have fun.

btw, It looks like py2app 0.3.6 has some problem with Qt 4.4.0 
instalation on my laptop(or with PyQt or SIP), so my attempt to create 
statically linked application bundle ends with some strange errors(there 
were no such errors with older versions of Qt,PyQt,SIP), so I have to 
start TreeLine 1.2.0 from commandline for now :-( but on the other hand, 
it runs much faster :-)
