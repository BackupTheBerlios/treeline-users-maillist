The TreeLine devlopment branch is finally released as stable!  You can
download the new 1.2.0 release at <http://treeline.bellz.org>.

Changes since 1.1.10 include an updated French translation and a few bug
fixes.  Bugs fixed include problems with good files being replaced with
bad backup files and fixes for path fields with spaces in the filenames.

The extensive changes since the last stable release (1.0.2) are listed on
the website.

Currently, the new release is only available in English and French.  As
always, volunteers to update translations for other languages are
welcome.

Doug.
