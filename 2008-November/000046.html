I really hate Windows DLL issues.  :(

I assume that 1.2.0 worked OK on this PC.  It's strange that you're
having problems, since the libraries haven't changed much.

Do you have any other installations of Python, Qt or PyQt on this PC?

Did you install 1.2.1 over an existing 1.2.0 install?  If so, you may
want to try removing both installs before re-installing 1.2.1.

You may also want to try temporarily moving or renaming the
"imageformats" directory that is under the "lib" directory.  The DLL's
in this folder were added to 1.2.1 to fix problems with support for some
image formats.

Let me know if any of this helps.
