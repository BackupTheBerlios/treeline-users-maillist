robin peterson wrote:
> Hi, is there a keyboard shortcut to switch from the left pane to the
> right pane?
> 
> I'm starting to use treeline; and I'm entering long sequences of nodes
> (siblings) that have multiple fields. I'd like to simply insert a
> sibling and start typing in the data editor pane, but when I press
> Ctrl A to add a node, it move the focus over to the left pane, and I
> have to use the mouse to move it back. I've tried all the obvious
> stuff (combinations of control keys with tab, enter, etc.), and I
> looked at the manual, but I'm not able to figure out how to do
> this...anybody know?

The tab key should do this, but there is a bug that often causes hidden
widgets to get the focus.  I'll fix this the next time I make a release.

A work-around that might help for now is to unset the general option
"Rename new nodes when created".  This will leave selection and focus
unchanged when nodes are added.
