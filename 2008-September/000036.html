Hi

I finally upgraded to the Qt4 version (1.2) of TreeLine on my Sony 
UX180.  I noticed that the Tools -> General Options dialog box is higher 
than what fits on my 1024x600 screen (with Start bar visable).  I can 
just barely click the Ok button if I move the window up as far as it 
will go.  Luckily the dialog box was not 1 pixel larger or I would not 
have been able to save ;-)

P.S. I finally upgraded because I was bitten by the 2 TreeLine programs 
running at the same time problem.
I had no idea I was running 2 TreeLines since only one appeared in my 
Start Tray.  The TreeLine window I was using to edit the file crashed.  
The TreeLine that was running by mistake obediently  wrote out it's 
backup ~trl file which was 8 days old. When I ran TreeLine it asked if I 
wanted to Restore and I said yes which of course restored from the old 
backup.
