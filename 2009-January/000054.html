Hi,

I just did another stable release to fix a few minor bugs and to add a
German tranlation.  There were also changes to the libraries used in the
Windows binary that will hopefully imporve stability.  You can get the
1.2.2 release from the usual place: <http://treeline.bellz.org>.

Here's the full changelog:

Updates:

 - Added German translations of the GUI and the ReadMe file.

 - Added a few missing strings to the translation files.

 - Substituted some imported modules to avoid deprecation warnings in
   Python 2.6. Note that Python 2.3 is no longer supported.

 - The windows binary has been updated to use Python 2.6, and the
   installer has been tweaked to reduce DLL conflicts.

Bug Fixes:

 - Fixed problems using the tab key to change focus from the tree to the
   data editor.

 - ID numbers are now properly assigned to new nodes with UniqueID
   fields in the title.

 - Fixed problems creating nodes with child count fields in the titles.


Doug.
