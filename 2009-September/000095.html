> Orbit is a GTK-related process communication system used by GTk/Gnome.
> It isn't used directly by TreeLine, but it is used by browsers like
> Firefox.  I suspect that TreeLine is trying to open a link using a
> "default" browser that isn't properly configured.
>One solution is to either properly configure the browser or to remove it
> from your system.  Another is to set the BROWSER environment variable
> for the user to the browser you do use (something like "opera %s").
