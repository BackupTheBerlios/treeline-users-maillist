Hi,

I want to warn users about a serious bug in TreeLine 1.3.1 (thanks to
Bruce Baum for reporting it).  When closing a window that contains a
modified file, TreeLine provides a save changes prompt.  Due to this
bug, TreeLine will not save the file in response to a "Yes" from the
user.  The window will close and the changes will be lost.

The work-around is to save the file before trying to close the window.

I'll fix this in the next release.

  Doug
