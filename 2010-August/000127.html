The ref="y" identifies a field as the link reference field.  That is the
field that internal links search when looking for a match.

By default, the link reference field is set to the first field in each
data type.  To change it to another field, click the "Show Advanced"
button in the "Type Config" tab of the Configure Data Types dialog.  A
pull-down selection will allow you to set the reference field to any of
the fields in a given data type.

If the link reference field is set to the InternalLink field, then the
links will find other nodes with similar links, until it cycles back to
the node where you started.  If it is set to a different field, then the
links will be one-way.
