Hi,

I've just released a new development release - 1.3.2.  This release
fixes a major bug that prevented files from being saved at the "Save
Changes?" dialog when closing a modified file.  The ReadMe documentation
was also updated to cover the new features added during this development
cycle.

Please keep in mind that this snapshot could be buggy.  It should not be
used for critical work.  Feedback is apprecisted.

It can be downloaded from
<http://treeline.bellz.org/download.html#unstable>.

I hope to keep this development cycle much shorter than the last one.  I
don't plan to include additional major changes until the next cycle.

More details are in the changelog:

Notes:
 - This is an unstable development snapshot. It may contain bugs.
   Testing and bug reports are appreciated, but the stable release
   (TreeLine 1.2.x) should be used for critical work.
 - The GUI and documentation translations have not yet been updated.

Updates:
 - Updated the ReadMe file to cover the new features added during this
   development cycle.
   
Bug Fixes:
 - Fixed a major bug that prevented files from being saved at the "Save
   Changes?" dialog when closing a modified file.
