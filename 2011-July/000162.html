> For my question, how does one do a search in the on-line help?

Are you using the latest version:  1.4.0?  Actually, the help viewer
search function is in versions 1.3.3 and newer, but I recommend using
the stable 1.4.0 release.

In 1.4.0, just type the text to find in the box near the top of the
Readme viewer.  The arrow buttons to the right can be used for next and
previous.
