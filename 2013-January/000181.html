Paul A. Crable wrote:
> I have my address book in Treeline.  I did the following:
> 
> Data -->  Add Category Level...
> 
>    I received the message "Cannot Expand Without Common Fields".
> 
> Could someone clarify what this means, how it might be generated, and
> what I need to do to add a category level without failure.

The add category level command adds an extra level of nodes based on the
content of some field(s) in child nodes.  It basically groups the
chldren by content that they have in common.

In order for this to work, all of the child nodes need to have at least
some fields with the same names.  In many cases, all of the children are
of the same data type, so they will have common field names.

You got that message because TreeLine couldn't find any field names that
all of the child nodes (children of the currently selected parent) had
in common.
